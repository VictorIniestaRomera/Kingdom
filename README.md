This is a personal project made whenever I had time.

This project works for me to test and learn more about C++ father/child class, organization while coding and thinking as a programmer.

The project is about a somewhat boardgame, made in Visual Studio console, representing a big land with four kingdoms in it.
Each player is the king of one of the kingdoms, and its interest is to conquer, and destroy the other kingdoms by building villages for
supplies and command outputs for new troops.
